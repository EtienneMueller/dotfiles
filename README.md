# dotfiles

This repo is for me personally to speed up setting up my Mac after resetting it.

## Steps

### 1. Run the bootstrap script from the terminal:

```zsh
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/EtienneMueller/dotfiles/main/bootstrap.sh)"
```
This will install:
- Xcode
- Homebrew
- Firefox and Obsidian casks

### 2. Log in to accounts

Log in to:
- Firefox + PW manager
- sync pCloud Documents folder

### 3. Run install script

- set up Git
- create symlinks
- download apps via homebrew
